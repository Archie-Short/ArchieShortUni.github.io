A junior games programming generalist with experience in unreal engine and C++

While all of my industry work is under NDA and I have greatly improved since, here are some of the projects I worked on before joining the industry over a year ago

ECHOES IN THE DARK

This was created for a game jam, made in UE4 I was the lead for this project handling all programming, graphics and most design. It is a puzzle game with an echolocation mechanic and has a heavy emphasis on sound.

COVERT CRAB

I created this for a games technology project in C++ with opengl, it's a wave shooter where you have to keep beacons active.

Thesis AI Project

A UE5 / C++ project that was my masters thesis, it is focused on guard AI for immersive sim games and how variation can be added to their behaviour without making them unpredictable and unfair.

BEAT DOWN

A UE5 / C++ bullet hell university project, the aim is to survive the song and a beatmapping structure is in place for easy level creation.

Wizard Roguelike Jam

A UE5 / C++ game jam where I got to create a level generator as well as data driven gameplay systems.
